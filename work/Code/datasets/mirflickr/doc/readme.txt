The images of the MIR Flickr collection are made available for research
purposes under the individual image licenses in the meta/license directory.
Please only redistribute the image collection as a whole, i.e. including the
meta directories.

For the most up to date information on this image collection, please visit:
http://press.liacs.nl/mirflickr. If you use the collection for scientific
reporting, please cite the following paper:

M.J. Huiskes, M.S. Lew (2008). ``The MIR Flickr Retrieval Evaluation'', ACM
International Conference on Multimedia Information Retrieval (MIR'08),
Vancouver, Canada.
